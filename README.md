# 🔐 Password Strength Checker

## 📌 Overview

This Python program checks the strength of a password and gives suggestions to improve it.
It also prevents reuse of old passwords using SHA-256 hashing.

---

## 🚀 Features

* Checks password strength (Weak / Medium / Strong)
* Validates length, uppercase, lowercase, numbers, special characters
* Detects common weak passwords
* Prevents password reuse
* Suggests stronger password
* Shows hashed password

---

## 🛠 Technologies

* Python
* `re` module
* `hashlib`

---

## ▶ How to Run

```bash
python password_checker.py
```

---

## 📊 Strength Levels

* Weak ❌
* Medium ⚠
* Strong ✅
